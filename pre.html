**AIM:** Write a program to show try-catch blocks.  
**REQUIREMENT:** JDK Compiler.

**// INPUT**
```java
import java.io.*;

class Example {
    public static void main (String args []) {
        FileInputStream fis = null;
        try {
            fis = new FileInputStream ("B:/my file.txt");
        }
        catch (FileNotFoundException fnfe) {
            System.out.println("The specified file is not" 
                + " present at the given path");
        }
        int k;
        try {
            while ((k = fis.read()) != -1) {
                System.out.print ((char) k);
            }
            fis.close ();
        }
        catch (IOException ioe) {
            System.out.println ("I/O error occured: " + ioe);
        }
    }
}
```

This program will display the file contents.

### Page 6: Experiment - 6(c)
**AIM:** Write a program to create multiple threads.  
**REQUIREMENT:** JDK Compiler.  
**THEORY:** Multiple threads can run concurrently in java, allowing better CPU utilization & multitasking.

**Code:**
```java
class Multithread extends Thread {
    public void run () {
        System.out.println ("Thread running: " + Thread.currentThread().getName());
    }

    public static void main (String [] args) {
        Multithread t1 = new Multithread ();
        Multithread t2 = new Multithread ();
        Multithread t3 = new Multithread ();

        t1.start ();
        t2.start ();
        t3.start ();
    }
}
```
**// Output**  
Thread running: Thread-0  
Thread running: Thread-1  
Thread running: Thread-2

### Page 7: Experiment - 5(b)
**AIM:** Write a program to demonstrate nested try blocks.  
**REQUIREMENT:** JDK Compiler.  
**THEORY:** A nested try block means a try block inside another try block. It is used when different parts of code may throw different exceptions.

**Code:**
```java
class NestedTryDemo {
    public static void main (String [] args) {
        try {
            int a = 10 / 2;
            try {
                int [] arr = {1, 2, 3};
                System.out.println (arr[5]);
            }
            catch (ArrayIndexOutOfBoundsException e) {
                System.out.println ("Inner catch: Array index error");
            }
        }
        catch (ArithmeticException e) {
            System.out.println ("Outer catch: Arithmetic error");
        }
        System.out.println ("Program continues normally");
    }
}
```
**// Output**  
Inner catch: Array index error  
Program continues normally
